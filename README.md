# homework-09

## Описание

Хук `useHover` отслеживает наведение курсора на элемент.

```js
const { hovered, ref } = useHover();
```

- hovered — true, если мышь над элементом.
- ref — нужно повесить на DOM-элемент.

Установка и запуск

```bash
npm install
npm run dev
```

## Основные файлы

- src/hooks/useHover.js — реализация.

- src/Demo.jsx — пример.
